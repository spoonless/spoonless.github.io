David Gayerie - Développeur && Formateur && Accompagnateur agile

Je forme les développeurs en entreprise mais également les étudiants et les personnes en reconversion professionnelle qui souhaitent travailler dans le numérique.

J'interviens au sein des écoles et des entreprises pour réaliser des formations sur les langages de programmation, les frameworks.

Depuis 2013, ils me font confiance : ....

Mes supports de formation en ligne :

Java

JavaScript

Python

Jakarta EE

Spring framework

Android

Ingénierie des tests

La formation est l'occasion pour moi de partager les connaissances et les savoir-faire de mon métier de développeur.
